针对gui for singbox遇到的问题
1. 修改大多设置都必须重启内核，托盘右键重启只是重启界面！
2. 添加规则没用，必须修改 配置 里面的规则集和规则！
3. 删除默认规则集，注意DNS配置里面也需要删除相关规则，这个破细节也折腾我很长时间
4. 更换机场，要么直接改原机场链接，要么就得改配置的出站设置。。。。真的麻烦
5. 默认配置必须改“漏网之鱼”，默认是走节点。符合大部分需求，缺点是一些国内网站容易跑代理，有封号风险。比如电商。。。。
他是个“好”软件，就是学习成本实在太高，只是想简单翻墙尽量别选
6. 这个软件的 配置 规则 订阅 设置都有两套规则，选中右键是一套设置，右上角三个点是另一套设置。。。
